About

Praylink is a service that allows people to text in a prayer request.

To get started, text "PRAY" to (979) xxx-xxxx

Now you are able to use all of the features that comes with Praylink.

To send in a prayer request, text "pray [prayer here]" for example: "pray that my mom feels better"

To update a prayer, text "update [prayer number] [update]" In order to find the prayer that you want to update, text 'update' and it will respond with the last 3 prayers you sent in that does not have an update. Example: "update 12 my mom feels better now"

This is just the surface of the features you can use, text 'commands' in order to get a full list of what you can do with the bot.
